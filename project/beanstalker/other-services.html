Route 53

The beanstalk:bind-domains will bind the current Elastic Load Balancer as an alias. In the example below, Route 53 Zone Id Z3DJ4DL0DIEEJA (for modafocas.org) will be assigned as services.modafocas.org:

$ mvn beanstalk:bind-domains -Dbeanstalk.domains=services.modafocas.org:Z3DJ4DL0DIEEJA

It is also possible to bind the root domain into it:

$ mvn beanstalk:bind-domains -Dbeanstalk.domains=modafocas.org
